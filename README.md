# ROB521 Labs - Mobile Robotics and Perception

A collection of laboratory assignments for the Mobile Robotics and Perception course (ROB521). This repository contains implementations for path planning, motion estimation, and mapping algorithms applied to mobile robotic systems.

## Course Overview

This course covers fundamental concepts in mobile robotics including:
- Path planning and navigation
- Robot motion estimation
- Sensor calibration and wheel parameter estimation
- Occupancy grid mapping

## Lab Contents

- **Lab 2**: Path following algorithms and trajectory planning
- **Lab 3**: Robot motion estimation, wheel radius/baseline calibration, and occupancy grid mapping
